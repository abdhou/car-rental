# Car Rental Management System

This project is a **simple car rental management application** designed to demonstrate the use of **Domain-Driven Design (DDD)**, **CQRS**, and **Event Sourcing (ES)** using **Spring Boot**.

The goal of this project is not to build a full-featured product, but to showcase **clean architecture**, **clear domain modeling**, and **modern backend design patterns**.

---

## Architecture Overview

The application follows a **DDD-inspired layered architecture** combined with **CQRS** and **Event Sourcing**.

### Key Concepts
- **DDD** for modeling the business domain
- **CQRS** to separate command and query responsibilities
- **Event Sourcing** to persist domain changes as immutable events
- **Spring Boot** as the application framework
